문자함수 연습

        0123456789012345678901234567890
        ABCDEFGHIJKLNMOPQRSTUVWXYZabcde

length : 객체(문자(열))의 길이

subString(num1,num2): 문자열 발췌

slice(num1,num2): 문자열 발췌

substr(num1,num2): 문자열 발췌

replace(a,b): a를 b로 첫번째만

replaceAll(a,b): a를 b로 모두

toUpperCase(): 대문자

toLowerCase(): 소문자

concat(a,b): 문자(열) 결합

trim(): 앞뒤 공백 제거

trimStart(): 앞쪽 공백 제거

trimEnd(): 앞쪽 공백 제거

문자열 공백 모두 : replaceAll

charAt(num): 문자열 위치 하나 발취

charCodeAt(num): 문자열 위치의 ASCII코드

split(): 특정문자를 기준으로 잘라낸다 이떄 잘라낸 값들은 배열로 기억

indexOf() 문자(열) 검색시 사용 : 있으면 위치값 없으면 -1반환

lastIndexOf() 문자(열)을 뒤에서 부터 검색시 사용 : 있으면 위치값 없으면 -1반환

serch() : 문자(열) 검색시 사용 : 있으면 위치값 없으면 -1반환

match() : 문자(열) 포함하고 있으면 그대로 반환 없으면 공백 반환

includes() : 문자(열)을 포함하고있으면 True 없으면 false
